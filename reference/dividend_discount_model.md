# Calculates discounted net profits based on a dividends discount model

Calculates discounted net profits based on a dividends discount model

## Usage

``` r
dividend_discount_model(data, discount_rate)
```

## Arguments

- data:

  A data frame containing the annual net profits on company technology
  level

- discount_rate:

  Numeric, that holds the discount rate of dividends per year in the
  DCF.
