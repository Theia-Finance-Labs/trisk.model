# Calculate baseline and transition shock trajectoroes

Calculate baseline and transition shock trajectoroes

## Usage

``` r
extend_assets_trajectories(trisk_model_input, start_year, shock_year)
```

## Arguments

- trisk_model_input:

  List with project agnostic and project specific input data

- start_year:

  Numeric, holding start year of analysis.

- shock_year:

  year of transition from baseline scenario to target scenario

## Value

A tibble holding annual profits
